Blend Shortcode plugin for WordPress

== Description ==

This plugin converts all Blend Shortcodes into embeddable Blend players. It works for any Blend project. Once you install this plugin, it works for any of your blog posts.

A simple example:

`[blend]https://blend.io/project/54611406124aec59070070c4[/blend]`

== Installation ==

1. Upload `blend-shortcode` to your plug-in directory or install it from the Wordpress Plug-in Repository
2. Activate the plugin through the 'Plugins' menu in WordPress
